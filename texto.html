¿Qué es HTML?

HTML: HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

Etiquetas HTML Básicas

Algunas etiquetas HTML básicas incluyen:

  • <html>: Define el comienzo y el final de un documento HTML.
  • <head>: Contiene metadatos sobre el documento, como el título de la página.
  • <body>: Contiene el contenido visible de la página web.
  • <p>: Define un párrafo de texto.
  • <h1> a <h6>: Encabezados de diferentes niveles.
